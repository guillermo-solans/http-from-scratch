Cats from Scratch

An HTTP/1.1 server built directly on top of raw TCP sockets — no HttpListener, no Kestrel, no ASP.NET. Just System.Net.Sockets and a lot of \r\n.

What is this?

You are talking to a tiny C# program that opens a TCP socket on a port, parses HTTP requests byte by byte, routes them, and writes responses with hand-built status lines and headers. The page you are reading was served by that same loop.

REST endpoints

GET/cats
GET/cats/:id
POST/cats — JSON body { name, breed, age }
PUT/cats/:id — JSON body
DELETE/cats/:id

Try it from the browser

Click any button to send a request.

Curl recipes

curl -i http://localhost:8080/cats
curl -i http://localhost:8080/cats/1
curl -i -X POST http://localhost:8080/cats \
     -H "Content-Type: application/json" \
     -d '{"name":"Simba","breed":"Bengal","age":4}'
curl -i -X PUT http://localhost:8080/cats/1 \
     -H "Content-Type: application/json" \
     -d '{"name":"Luna","breed":"Siberian","age":4}'
curl -i -X DELETE http://localhost:8080/cats/3
Served on a hand-rolled TCP listener. Concurrency by async/await.
